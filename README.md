# 🩺 Diabetes Prediction Using Machine Learning

## 📌 Project Overview
This project focuses on predicting diabetes using machine learning techniques based on medical features such as glucose level, BMI, age, blood pressure, and other health parameters.

The aim is to build an accurate model for early detection of diabetes using Data Science techniques.

## 📊 Dataset
- Source: Kaggle Pima Indians Diabetes Dataset  
- Features: Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Age  
- Target: Outcome (0 = No Diabetes, 1 = Diabetes)

---

## ⚙️ Data Preprocessing
- Handling missing values (replace zeros with mean/median)  
- Outlier detection and removal  
- Feature scaling (Standardization)  
- Train-test split (80/20)

---

## 📈 Exploratory Data Analysis (EDA)
- Histograms  
- Box plots  
- Correlation heatmap  
- Scatter plots  

---

## 🧠 Feature Engineering
- BMI category creation  
- Feature selection using correlation  
- Removal of less important features  

---

## 🤖 Machine Learning Models
- Logistic Regression  
- Decision Tree  
- Random Forest  

---

## 📊 Model Evaluation
- Accuracy Score  
- Precision  
- Recall  
- F1 Score  
- Confusion Matrix  

Random Forest performed the best among all models.

---

## 🏁 Conclusion
This project demonstrates how Data Science and Machine Learning can help in early prediction of diabetes. Glucose level and BMI are found to be the most important factors affecting diabetes risk.
